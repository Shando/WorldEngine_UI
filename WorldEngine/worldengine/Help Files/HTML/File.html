File Menu


This is a fairly standard File Menu, so I'll only point out a couple of options.



Save Current Map View        This allows the saving of the current view of the Map as displayed in the Main Viewport of the UI.

Print Current Map View        This allows the printing of the current view of the Map as displayed in the Main Viewport of the UI.

Set Output Directory        This allows the setting of the Directory to be used for ALL output (world, maps etc.).
