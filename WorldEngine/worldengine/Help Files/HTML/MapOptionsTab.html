Map Options


The Map Options tab consists of two parts. The uppermost part is what I call the MiniMap, which shows a smaller version of the Main Viewport. The main difference between the MiniMap and the Main Viewport (besides their respective sizes!) is that the MiniMap will ALWAYS show the whole Map, whereas the content of the Main Viewport can be zoomed in, moved, zoomed out etc.



The red rectangle visible in the MiniMap shows the area of the Map that is currently displayed in the Main Viewport.


The lower part of this tab displays the various parameters for the various Simulations. More information on these can be found here.
