Map Parameters


The Map Parameters part of the Map Options tab contains tabs for all of the Simulations that have parameters that can be changed:


  • Erosion
  • Humidity
  • Hydrology
  • Ice Caps
  • Irrigation
  • Permeability
  • Precipitation
  • Temperature
  • Wind
